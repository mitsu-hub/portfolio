Machine Learning–Based Intrusion Detection & Prevention System Evaluation

Kanto Gakuin University | Sep 2022 – Mar 2024

This project represents a two-year research effort that significantly shaped my interest in cybersecurity and my long-term career goals. Driven by a strong interest in Intrusion Detection and Prevention Systems (IDPS), I conducted an in-depth study to understand how machine learning can be applied to real-world network security challenges. Using the CICIDS2017 dataset, I implemented and evaluated multiple machine learning algorithms to analyze their effectiveness in detecting malicious network traffic. Through this process, I moved from a surface-level understanding of IDPS concepts to a deep, practical understanding of both their capabilities and limitations.

  • Implemented and evaluated four machine learning algorithms: Support Vector Machine (SVM), Naive Bayes, Decision Tree, and k-Nearest Neighbors
  • Conducted binary classification experiments focusing on detection accuracy using precision and recall as primary evaluation metrics
  • Developed extensive Python-based implementations, writing approximately 10 source files with over 1,000 lines of code each
  • Identified gaps between theoretical expectations and real-world model performance, particularly in false positives for normal traffic
  • Iteratively refined experiments through continuous trial and error, dedicating significant time to research and experimentation

Through this research, I gained a realistic perspective on the challenges of deploying machine learning–based security systems in operational environments. The experience fundamentally changed how I view cybersecurity, shifting my focus toward detection accuracy, operational impact, and analyst usability. This project directly influenced my aspiration to pursue a future role in a Security Operations Center (SOC), where accurate detection and informed decision-making are critical.

Technologies: Machine Learning, Intrusion Detection, Python, CICIDS2017, Network Security
